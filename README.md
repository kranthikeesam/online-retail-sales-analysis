# Online Retail Sales Analysis

## Overview
This project analyzes online retail transaction data to understand revenue trends and top-performing countries.  
It demonstrates practical data analyst skills in data cleaning, exploratory data analysis (EDA), and visualization using Python.

## Dataset
- Source: UCI Machine Learning Repository – Online Retail Data Set
- Description: Transactions from a UK-based online retail store between 2010–2011.
- File used: `data/Online_Retail.xlsx`

## Questions Answered
- How does monthly revenue change over time?
- Which countries contribute the most revenue?

## Tech Stack
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
- Git & GitHub
